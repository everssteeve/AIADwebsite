## Vue d'ensemble du processus

Le processus AIAD s'organise en 5 phases principales :

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   PHASE 1    │────▶│   PHASE 2    │────▶│   PHASE 3    │
│INITIALISATION│     │ PLANIFICATION│     │DÉVELOPPEMENT │
│  (3-7 jours) │     │  (2-4 heures)│     │(80% itération)│
└──────────────┘     └──────────────┘     └──────┬───────┘
                                                  │
                     ┌──────────────┐     ┌──────▼───────┐
                     │   PHASE 5    │◀────│   PHASE 4    │
                     │ DÉPLOIEMENT  │     │  VALIDATION  │
                     └──────────────┘     └──────────────┘
```

## Comment utiliser ce guide

1. **Nouveau projet** : Commencez par la Partie 0 (Préambule) puis suivez la Partie 1 (Initialisation)
2. **Itération en cours** : Consultez les Parties 2 à 5 selon votre phase actuelle
3. **Amélioration continue** : Utilisez la Partie 6 pour vos rituels d'équipe
4. **Référence rapide** : La Partie 7 (Annexes) contient les commandes et FAQ
