# PARTIE 7 : ANNEXES

## 7.1 Commandes Claude Code

| Commande | Description |
|----------|-------------|
| /clear | Nettoie le contexte (avant chaque tâche) |
| /context | Affiche l'utilisation du contexte |
| /rewind | Annule les dernières actions |
| /model | Change de modèle (Haiku, Sonnet, Opus) |
| /init | Initialise CLAUDE.md |
| /agents | Gère les SubAgents |
| Esc | Interrompt Claude |
| Shift+Tab | Change le mode d'acceptation |

## 7.2 Troubleshooting

| Problème | Solution |
|----------|----------|
| Claude ne répond pas | /clear puis réessayer |
| Code hors sujet | Reformuler SPEC avec exemples |
| MCP ne fonctionne pas | `claude mcp list` pour vérifier |
| SubAgent pas trouvé | Vérifier .claude/agents/ |
